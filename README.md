# Holly Arora Portfolio

Single-page static portfolio website for Holly Arora.

## Local Preview

Open `index.html` directly in a browser, or run a simple static server:

```bash
python3 -m http.server 4173
```

## Deployment

This repo includes a GitHub Pages workflow at `.github/workflows/pages.yml`. Pushes to `main` deploy the static site artifact from the repository root.
